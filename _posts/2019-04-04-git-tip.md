Git 을 터미널로 사용한다던지해서 한다면 계정이름, 비밀번호를 계속 입력하게 될 것이다.
그럴때는 아래와 같은 명령어를 입력하면 모든 저장소에서 인증을 생략해도 될 것이다. (최초의 인증은 필요)
유저 인증 정보를 git 설정에서 저장한다고 한다.
```
git config credential.helper store --global
```
